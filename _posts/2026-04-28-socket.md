socket相关函数：
/**
 * @brief 创建通信端点（套接字）
 * @param domain 协议族/地址族
 *     - AF_INET: IPv4协议
 *     - AF_INET6: IPv6协议  
 * @param type 套接字类型
 *     - SOCK_STREAM: 面向连接的TCP套接字（可靠，双向字节流）
 *     - SOCK_DGRAM: 无连接的UDP套接字（不可靠，数据报文）
 * @param protocol 具体协议类型,通常设为0,由前两个参数自动选择
 * @return 成功返回非负文件描述符，失败返回-1
 */
int socket(int domain, int type, int protocol);
